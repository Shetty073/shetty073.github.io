# Ashish Shetty - Portfolio

Welcome to the source code for my personal portfolio! This is a modern single-page application built with React, Vite, TypeScript, and Tailwind CSS. It highlights my professional work as a Tech Lead, my side projects, and my GitHub repositories.

The project automatically fetches my open-source repositories from the GitHub API and presents them in a beautiful, responsive layout with Framer Motion animations.

## Tech Stack
- **React** (v19)
- **Vite** (Build Tool)
- **TypeScript**
- **Tailwind CSS** (v4)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Development Setup

The source code for the React application is situated in the `source` directory.

### Running Locally
To run the development server:

```bash
cd source
npm install
npm run dev
```
Then, open your browser and navigate to `http://localhost:5173`.

### Building for Production
To build the project:

```bash
cd source
npm run build
```

This will bundle the React application and output the final `index.html` and `assets` folder directly into the *root directory* of this repository so it can be automatically hosted by GitHub Pages.
